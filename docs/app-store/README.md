# Eightful - App Store submission pack

Everything Apple asks for when you hit "Create new app" in App Store Connect,
drafted but not submitted. Review, tweak, paste.

- [description.md](description.md) - listing copy (name, subtitle, promo, long description)
- [keywords.md](keywords.md) - 100-character keyword string + reasoning
- [privacy-policy.md](privacy-policy.md) - privacy policy (HealthKit apps MUST have a hosted URL)
- [review-notes.md](review-notes.md) - reviewer notes pre-empting the trademark/HealthKit questions
- [nutrition-label.md](nutrition-label.md) - privacy nutrition label answers
- [screenshot-checklist.md](screenshot-checklist.md) - what you need to capture and at which device sizes
- [assets-checklist.md](assets-checklist.md) - icons, preview videos, category, rating

Host the privacy policy at a stable URL before submitting. Everything else
goes into App Store Connect directly.
